# Miftāḥ مِفْتَاح

**Prototyp eines erklärend-enzyklopädischen Wörterbuchs kulturgebundener islamischer Schlüsselbegriffe für deutschsprachige Konvertierte**

Begleitprototyp zur Masterarbeit von Ahmed Shaita (2026).

## Über das Projekt

Miftāḥ („Schlüssel") ist die praktische Umsetzung der in Kapitel 5 der Masterarbeit entwickelten Konzeption eines hybriden erklärend-enzyklopädischen Wörterbuchs. Der Prototyp demonstriert die drei lexikographischen Strukturebenen (Makro-, Mikro- und Mediostruktur) und das modulare Eintragsmodell mit acht Modulen.

## Aktuelle Version

Diese Vorschau enthält ein einzelnes Lemma (*Taqwā*) als Demonstration. Die vollständige Version mit sechs Lemmata (*Taqwā*, *Zakāt*, *Īmān*, *Ṣadaqa*, *Ṣalāh*, *Dīn*) folgt.
